# **App Name**: HearthLink

## Core Features:

- Anonymous Authentication: User authentication using Firebase anonymous sign-in.
- Real-time Collaboration: Real-time synchronization of annotations, highlights, and reading progress.
- PDF Handling and Rendering: Local PDF processing and rendering using PDF.js with page navigation and zoom controls.
- Advanced Annotation System: Create and manage annotations with precise positioning, user attribution, and color coding.
- Intelligent Highlighting System: Highlight text with user-specific colors and manage overlapping highlights.
- Real-time Chat System: Text-based chat for in-room communication, notifications for user actions.
- Smart annotation placement: AI-powered tool which detects important keyphrases or sections within the book, making it easy to create annotations.

## Style Guidelines:

- Primary color: Amber (#D2691E) evokes warmth and comfort.
- Background color: Warm off-white (#FFF8DC) creates a cozy atmosphere.
- Accent color: Ember red (#DC143C) for interactive elements and notifications.
- Body font: 'Source Sans Pro' (sans-serif) for readability of extended text.
- Headline font: 'Playfair' (serif) for headers.
- Use custom fireplace-themed icons.
- Subtle animations for page turns and loading states with warm transitions.
